# Vue代码实时预览

这是一个基于Vue 3、Vite、TypeScript、UnoCSS和Monaco Editor的代码实时预览应用。允许用户在左侧编辑HTML代码，并在右侧实时查看渲染结果。

## 技术栈

- Vue 3 - 前端框架
- TypeScript - 类型安全的JavaScript
- Vite - 现代构建工具
- UnoCSS - 原子化CSS框架
- Monaco Editor - 代码编辑器（VSCode的编辑器组件）

## 功能

- 实时HTML代码编辑与预览
- 美观的深色主题编辑器
- 自动应用UnoCSS样式到预览内容
- 响应式设计

## 本地开发

```bash
# 安装依赖
pnpm install

# 启动开发服务器
pnpm dev

# 构建生产版本
pnpm build
```
